Projeto 02

A partir do segundo projeto, começamos a usar o tinkercad, para projetar circuitos em arduinos. E o primeiro que fizemos foi de um semáforo.

Componentes utilizados:

  1. Arduino Uno R3:
    • 14 entradas e saídas digitais, no qual 6 pinos podem ser utilizados como saída PWM, que funciona como se adicionássemos intensidade no sinal. Dessa forma, ela pode ser utilizada, por exemplo, como volume, para regular a intensidade de brilho e muito mais.
  2. Placa de Ensaio Pequena:
    • É uma placa com furos e conexões condutoras ultilizada para a montagem de protótipos e projetos em estado inicial.
  3. Resistores:
    • A presença de um resistor de maneira isolada em um circuito tem a função de limitar a passagem de corrente.
  4. LEDs:
    • Possuem a capacidade de transformar energia em luz.

    Nele usamos o seguinte código:
    // C++ code // #define vm 13
    #define am 12
    #define vd 11
    int tempoVm = 3000;
    int tempoAm = 1000;
    int tempoVd = 2000;

    void setup()
    {
    pinMode(vm, OUTPUT);
    pinMode(am, OUTPUT);
    pinMode(vd, OUTPUT);

    }

    void loop()
    {
    digitalWrite(vm, HIGH);
    delay(tempoVm); // Wait for 1000 millisecond(s)
    digitalWrite(vm, LOW);
    digitalWrite(vd, HIGH);
    delay(tempoVd); // Wait for 1000 millisecond(s)
    digitalWrite(vd, LOW);
    digitalWrite(am, HIGH);
    delay(tempoAm);
    digitalWrite(am, LOW);
    }
